Heroku app: https://jokeservice69.herokuapp.com/
- Start med at vælge en jokeservice side.
- Hvis valgte side har oprettet deres end points korrekt, kan man slette/rette og poste jokes til valgte side.
- Vores egen side hedder: Dad jokes (Her kan du fx teste slet/ret og post funktionerne uden problemer).
- Der er brugt contenteditable på html elementerne der holder setup og punchline, så det er muligt at rette direkte i teksten i browseren, selvom det ikke ligger i et input felt.
